# [Live Trains UK](https://live-trains-uk.vercel.app/)

## A web application for finding live information about UK rail

A mobile-first app built with Next.js and UK Rail APIs for viewing live, real-time information about UK rail. You can access the site [here](https://live-trains-uk.vercel.app/).

Functionality and features include:

- Finding live direct departures between 2 stations
- Finding multi-leg journey plans between 2 stations
- Clear UI for distinguishing between fast and slow services
- Login with Google for naming and saving journeys
- Real-time updates on alerts and disruptions across the UK rail network
